🦷 Dental SLAM Tracker

📌 Project Summary

This research project tracks 3D dental surface changes over time using ICP-based registration and SLAM-like pose graph optimization. It simulates and quantifies wear progression, offering a novel approach for cavity and erosion detection using public datasets.

⚙️ Key Features

  • ICP Alignment (Pairwise and Global)
  • Pose Graph SLAM with Loop Closures
  • Artificial Wear Simulation
  • Curvature and Hausdorff Distance Tracking
  • Multi-patient batch runner

🖼️ Screenshots / Mesh Previews

See the repo for screenshots and visualizations of aligned and worn meshes with color maps.

🚀 How to Run

  1. Clone the repo: git clone https://github.com/Muhammedswalihu/dental-slam-tracker
  2. Install dependencies: pip install -r requirements.txt
  3. Run the pipeline: python dental_wear_pipeline.py
